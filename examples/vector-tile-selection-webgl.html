--- layout: example.html title: Vector Tile Selection (WebGL) shortdesc: Select features from vector tiles rendered with WebGL. docs: >

Click a rendered vector-tile feature to highlight it on the map. Click on an empty spot (ocean) to reset the selection. By changing the action type to "Multi Select" you can select multiple features at a time. With "Single Select on hover", features will be highlighted when the pointer is above them.

This example uses WebGL rendering for the vector tiles. Hit detection on the WebGL layer identifies the clicked feature, and a canvas-based VectorTileLayer sharing the same source highlights all instances of the selected feature across tile boundaries using renderMode: 'vector'.
