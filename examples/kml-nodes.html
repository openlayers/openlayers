--- layout: example.html title: KML with a styled nodes shortdesc: Rendering KML with a styled nodes. docs: >

This example uses the ol/format/KML to parse KML for rendering with a vector source.

If IconStyle for LineString is defined then this icon is used for all nodes of LineString o Polygon.

When IconStyle/Icon/href contains 'rotate' icon of all nodes will be rotated by tangent of segment.
