--- layout: example.html title: WebGL Tile Transitions shortdesc: Example of smooth tile transitions when changing the dimension of a WebGL layer. docs: > Demonstrates smooth reloading of layers when changing a dimension continuously. The reason this example exists is to show this feature working with the WebGL based renderer as compared to the default canvas based renderer shown in the WMTS Tile Transitions example. The demonstration layer is a global sea-level computation (flooding computation from SCALGO, underlying data from CGIAR-CSI SRTM) where cells that are flooded if the sea-level rises to more than x m are colored blue. The user selects the sea-level dimension using a slider. tags: "webgl, parameter, transition, grid" ---
